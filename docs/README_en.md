😊 MAA Assistant for NTE 😊  
Powered by MaaFramework!

> This project is in early development. PRs and Issues are welcome!

## ✨ Features
- 🎣 Auto Fishing
- 🥤 Auto Coffee Brewing
- 💰 Auto Extract Café Revenue

## 💡 Notes
- The game must run at **1280×720 resolution** in **windowed mode**
- Auto fishing mode is recommended at **30 FPS**
- Auto coffee brewing requires setting mouse input to **Seize** mode — this will temporarily take over your mouse
